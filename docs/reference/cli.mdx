# CLI Reference

The QUANTA CLI provides command-line access to all network operations.

## Installation

```bash
pip install quanta-cli
```

Verify installation:

```bash
quanta --version
```

## Global Options

| Option | Description |
|--------|-------------|
| `--network` | Network to use: mainnet, testnet (default: mainnet) |
| `--wallet` | Wallet name to use |
| `--config` | Path to config file |
| `--verbose`, `-v` | Enable verbose output |
| `--json` | Output in JSON format |
| `--help`, `-h` | Show help |

## Commands

### wallet

Wallet management commands.

#### wallet create

Create a new wallet.

```bash
quanta wallet create --name my_wallet
```

**Options:**
- `--name` (required): Wallet name

**Output:**
```
Creating new wallet...
Wallet created successfully!

Address: 5GrwvaEF5zXb26Fz9rcQpDWS57CtERHpNehXCPcNoHGKutQY
Mnemonic: word1 word2 word3 ... word24

⚠️  Write down your mnemonic phrase and store it safely!
```

#### wallet list

List all wallets.

```bash
quanta wallet list
```

**Output:**
```
Wallets:
  my_wallet     5GrwvaEF...  (default)
  test_wallet   5FHneW46...
```

#### wallet balance

Check wallet balance.

```bash
quanta wallet balance --wallet my_wallet
```

**Output:**
```
Wallet: my_wallet (5GrwvaEF...)

Balances:
  TAO:           125.50
  α-token:       5,000.00
  α-token (staked): 3,000.00
```

#### wallet export

Export wallet for backup.

```bash
quanta wallet export --wallet my_wallet --output backup.json
```

#### wallet import

Import wallet from backup.

```bash
quanta wallet import --input backup.json
```

### signal

Signal management commands.

#### signal submit

Submit a new signal.

```bash
quanta signal submit \
  --wallet my_wallet \
  --file portfolio.json \
  --stake 15 \
  --tag "momentum_v1"
```

**Options:**
- `--wallet` (required): Wallet to use
- `--file` (required): Portfolio JSON file
- `--stake` (required): α-tokens to stake
- `--tag`: Strategy tag

**Portfolio file format:**
```json
{
  "AAPL": 0.10,
  "MSFT": 0.08,
  "NVDA": 0.12
}
```

#### signal status

Check signal status.

```bash
quanta signal status sig_abc123
```

**Output:**
```
Signal: sig_abc123
Status: ACTIVE
Wallet: 5GrwvaEF...
Stake: 15.00 α

Performance (7-day):
  Return:       +2.3%
  Sortino:      1.85
  Calmar:       1.62
  Max Drawdown: 3.2%
  Rank:         45/230

Rewards (pending):
  TAO:     0.15
  α-token: 2.30
```

#### signal list

List your signals.

```bash
quanta signal list --wallet my_wallet
```

**Options:**
- `--wallet`: Filter by wallet
- `--status`: Filter by status (active, pending, closed)
- `--limit`: Max results

#### signal update

Update signal portfolio.

```bash
quanta signal update sig_abc123 \
  --file new_portfolio.json \
  --wallet my_wallet
```

#### signal withdraw

Withdraw signal and stake.

```bash
quanta signal withdraw sig_abc123 --wallet my_wallet
```

```bash

```

## Configuration File

Create `~/.quanta/config.yaml`:

```yaml
# Default network
network: mainnet

# Default wallet
default_wallet: my_wallet

# API settings
api:
  timeout: 30
  retries: 3

# Logging
logging:
  level: INFO
  file: ~/.quanta/logs/cli.log

# Submission settings
submission:
  auto_reveal: true
  default_stake: 10
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `QUANTA_NETWORK` | Default network |
| `QUANTA_WALLET` | Default wallet name |
| `QUANTA_API_KEY` | API key for authenticated requests |
| `QUANTA_CONFIG` | Path to config file |

## Examples

### Submit Signal Workflow

```bash
# 1. Check balance
quanta wallet balance --wallet trading

# 2. Check universe for tickers
quanta universe check AAPL MSFT NVDA

# 3. Submit signal
quanta signal submit \
  --wallet trading \
  --file my_portfolio.json \
  --stake 15 \
  --tag "momentum_v2"

# 4. Monitor performance
quanta signal status sig_abc123

# 5. Check leaderboard position
quanta leaderboard | grep sig_abc123
```

### Automated Script

```bash
#!/bin/bash
# daily_submission.sh

# Generate portfolio (your logic)
python generate_signal.py > /tmp/portfolio.json

# Submit to QUANTA
quanta signal submit \
  --wallet trading \
  --file /tmp/portfolio.json \
  --stake 15 \
  --tag "daily_momentum" \
  --json > /tmp/result.json

# Log result
cat /tmp/result.json >> ~/quanta_submissions.log
```

## Troubleshooting

### Common Issues

```bash
# Connection errors
quanta --network testnet epoch status  # Test connectivity

# Authentication errors
quanta wallet list  # Verify wallet exists

# Verbose debugging
quanta -v signal submit ...
```
