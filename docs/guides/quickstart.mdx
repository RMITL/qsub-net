# Quickstart Guide

This guide gets you from zero to submitting your first signal in about 10 minutes.

## Prerequisites

Before you begin, you'll need:

- [ ] Python 3.9+ installed
- [ ] Basic understanding of stock trading concepts
- [ ] Small amount of α-tokens (for testnet, use faucet)

<Note>
**New to crypto?** You'll need a wallet to interact with QUANTA. We'll cover wallet setup below.
</Note>

## Step 1: Install QUANTA CLI

<Tabs>
  <Tab title="pip (Recommended)">
    ```bash
    pip install quanta-cli
    ```
  </Tab>
  <Tab title="From Source">
    ```bash
    git clone https://github.com/RMITL/QUANTA
    cd QUANTA
    pip install -e .
    ```
  </Tab>
</Tabs>

Verify installation:

```bash
quanta --version
# quanta-cli v1.0.0
```

## Step 2: Create a Wallet

Create a new wallet for QUANTA:

```bash
quanta wallet create --name mywallet
```

Output:
```
Creating new wallet...
Wallet created successfully!

Address: 5GrwvaEF5zXb26Fz9rcQpDWS57CtERHpNehXCPcNoHGKutQY
Mnemonic: [SAVE THIS SECURELY - NEVER SHARE]

⚠️  Write down your mnemonic phrase and store it safely!
```

<Warning>
**CRITICAL:** Write down your mnemonic phrase on paper and store it securely. This is your only backup. Never share it with anyone.
</Warning>

## Step 3: Get Testnet Tokens

For testnet, use the faucet:

```bash
quanta faucet request --wallet mywallet --network testnet
```

Output:
```

✓ Received 100 α (testnet)
Balance: 100 α
```

## Step 4: Create Your First Signal

Create a simple momentum signal:

```python
# signal.py
from quanta import SignalPool, Wallet

# Load wallet
wallet = Wallet.load("mywallet")

# Define portfolio (weights must sum to ≤ 1.0 for long-only)
portfolio = {
    "AAPL": 0.15,   # 15% Apple
    "MSFT": 0.12,   # 12% Microsoft
    "NVDA": 0.10,   # 10% Nvidia
    "GOOGL": 0.08,  # 8% Google
    "AMZN": 0.08,   # 8% Amazon
    "META": 0.07,   # 7% Meta
    "TSLA": 0.06,   # 6% Tesla
    "BRK.B": 0.05,  # 5% Berkshire
    "UNH": 0.05,    # 5% UnitedHealth
    "JPM": 0.04,    # 4% JPMorgan
}

# Connect to testnet
pool = SignalPool(network="testnet")

# Submit signal
result = pool.submit_signal(
    portfolio=portfolio,
    wallet=wallet,
    stake_amount=10,  # 10 α stake
    strategy_tag="momentum_v1"
)

print(f"Signal submitted!")
print(f"Signal ID: {result.signal_id}")
print(f"Status: {result.status}")
```

Run it:

```bash
python signal.py
```

Output:
```
Signal submitted!
Signal ID: sig_abc123def456
Status: COMMITTED
```

## Common Issues

<AccordionGroup>
  <Accordion title="'No module named quanta' error">
    Make sure you installed the package:
    ```bash
    pip install quanta-cli
    ```

    If using a virtual environment, ensure it's activated.
  </Accordion>

  <Accordion title="'Insufficient balance' error">
    You need α-tokens to stake. For testnet:
    ```bash
    quanta faucet request --wallet mywallet
    ```

    For mainnet, acquire α-tokens from a DEX.
  </Accordion>

  <Accordion title="'Invalid ticker' error">
    Your portfolio contains a ticker not in the approved universe. Check valid tickers:
    ```bash
    quanta universe list
    ```
  </Accordion>

  <Accordion title="'Commit window closed' error">
    You missed the submission window. Windows open every hour:
    ```bash
    quanta epoch status
    # Shows next commit window
    ```
  </Accordion>
</AccordionGroup>
