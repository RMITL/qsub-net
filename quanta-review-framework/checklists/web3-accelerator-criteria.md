## Application Quality Checklist

### Written Materials
- [ ] Clear, concise executive summary
- [ ] Problem statement compelling
- [ ] Solution clearly explained
- [ ] Technical depth appropriate
- [ ] Financials realistic
- [ ] Team strengths highlighted
- [ ] Ask clearly stated

### Presentation/Pitch
- [ ] Story flows logically
- [ ] Demo/prototype shown
- [ ] Questions anticipated
- [ ] Passion evident
- [ ] Coachable attitude

- [ ] One-pager

- [ ] Financial model
- [ ] Team bios

## Red Flags Accelerators Look For

### Immediate Disqualification
- [ ] Anonymous team
- [ ] Copied project
- [ ] Clear security vulnerabilities
- [ ] Regulatory non-compliance
- [ ] Unrealistic claims
- [ ] No technical substance

### Concerns (May Still Consider)
- [ ] Solo founder
- [ ] Part-time commitment
- [ ] No prior startup experience
- [ ] Early stage (concept only)
- [ ] Crowded market
